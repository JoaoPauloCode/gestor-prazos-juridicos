# ⚖️ Gestor de Prazos Processuais (Python)

Sistema em Python para organização e controle de processos jurídicos, com cálculo de prazos conforme o Novo CPC.

## Base legal

Art. 224 do CPC:
- Exclui o dia do começo
- Inclui o dia do vencimento
- Contagem em dias úteis

## Funcionalidades

- Cadastro de processos
- Cálculo de prazos em dias úteis
- Ordenação por prazo
- Identificação de urgência
- Busca por nome
- Validação de datas
- Persistência em JSON

## Tecnologias

- Python 3
- JSON

## ▶️ Como executar

```bash
python main.py
```

## Diferencial

Projeto que conecta Direito e Tecnologia, simulando um sistema real de controle de prazos jurídicos.
